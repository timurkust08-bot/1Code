Основной функционал программы включает интерактивное текстовое меню, локальное хранение данных, валидацию пользовательского ввода и контроль дубликатов. Пользователь может взаимодействовать с системой с помощью команд меню от 1 до 6. Программа автоматически проверяет корректность вводимых данных (запрет пустых строк, валидация оценки в диапазоне от 1 до 5), сверяет новые записи с уже существующими для предотвращения дублирования по связке автор и название, а также рассчитывает аналитические метрики в реальном времени.
В состав проекта входят четыре файла. main.py является основным исполняемым файлом и содержит всю программную логику, включая интерфейс, обработку данных и математические вычисления. books.json выступает в роли локальной базы данных, где информация хранится в структурированном текстовом формате JSON в виде списка словарей. .gitignore представляет собой служебный файл для системы контроля версий Git, который указывает на необходимость игнорирования системного кэша и папок виртуального окружения. README.md содержит текстовую документацию, техническое описание проекта и руководство по его запуску.
Архитектура кода main.py состоит из набора функций, разделенных по зонам ответственности.
Системные функции load_books и save_books отвечают за работу с файлами. load_books загружает информацию из books.json и преобразует ее в список Python, обрабатывая отсутствие файла или ошибки чтения путем возврата пустого списка. save_books записывает текущее состояние списка книг обратно в файл books.json с сохранением структуры и кодировки текста.
Функции бизнес-логики реализуют непосредственные пункты меню. add_book отвечает за сбор информации о новой книге, проведение проверок на дубликаты без учета регистра букв и сохранение записи. show_all_books выводит на экран пронумерованный перечень всех сохраненных книг с указанием автора, названия, оценки и даты прочтения. show_average_rating производит расчет среднеарифметического значения оценок по всем книгам и выводит результат с округлением до двух знаков после запятой. show_author_stats группирует имеющиеся данные по авторам и подсчитывает суммарное количество прочитанных книг для каждого из них. delete_book предоставляет пользователю выбор: удалить запись по ее порядковому номеру в списке либо путем ввода точного имени автора и названия книги.
Управляющая функция main запускает бесконечный цикл, который отображает текстовое меню в консоли, считывает выбор пользователя, вызывает соответствующую целевую функцию и завершает работу приложения при вводе команды выхода.
